# diary

A small tool for managing my digital diary.

Currently its only purpose is categorizing content, i.e. it transforms a flat
list of files:

``` 
IMG_1234.jpg
IMG_1235.jpg
IMG_1236.jpg
```

... into a per-day tree:

```
2018/01/01/12-34-56.jpg
2018/01/02/21-37-00.jpg
2018/01/03/16-35-00.jpg
```
